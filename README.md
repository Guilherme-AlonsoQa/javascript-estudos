# JavaScript - Estudos

Repositorio de estudos da Formacao JavaScript Developer, parte da minha trilha de desenvolvimento Fullstack. Aqui registro anotacoes, exercicios e projetos praticos enquanto domino a linguagem.

## Topicos estudados

Variaveis, tipos e operadores; estruturas de controle e funcoes; arrays e objetos; manipulacao do DOM; eventos; ES6+ (arrow functions, destructuring, modules); programacao assincrona (Promises, async/await); consumo de APIs com fetch; e boas praticas.

## Estrutura do repositorio

A pasta exercicios contem exercicios de logica e linguagem, a pasta projetos contem aplicacoes praticas e a pasta anotacoes contem resumos de estudo.
